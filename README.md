# [Markov Chain Visualizer](https://rkee.github.io/markov-chain-visualizer/)

Visualization tool for simulating and analyzing any finite markov chains.

## Features:
* Directed graph editing tools
* Stochastic matrix
* Reducibility
* Periodicity
