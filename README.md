# Markov Chain Visualizer

## About
Visualize and analyze any finite markov chain with this tool!

## Supported features:
* Directed graph editing tools
* Stochastic matrix visualization
* Reducibility
* Periodicity

## [Live tool] (https://rkee.github.io/markov-chain-visualizer/)
